This project is a Traffic Data Warehouse designed to demonstrate core Data Engineering, SQL, and Analytical Processing concepts.
It simulates how real-world traffic systems collect, clean, store, and analyze high-volume data for insights and decision-making.

The aim is to build an end-to-end analytical environment that includes:
Data ingestion
Data cleaning & preprocessing
Dimensional modeling (Fact & Dimension tables)
OLAP-style analytical queries
Future integration with Power BI / Tableau

---
Project Objectives

Design a clean, well-structured SQL data warehouse
Transform raw traffic logs into analytical-ready tables
Implement ETL logic inside SQL
Perform traffic analytics using optimized SQL queries
Enable dashboarding tools to consume processed data
